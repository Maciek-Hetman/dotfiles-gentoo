# dotfiles-gentoo
Config for my PC with Ryzen 3 3200G (--march=znver2)  
General Gentoo setup:  
    - systemd  
    - desktop/gnome/systemd profile  
    - GNOME packages with accept_keywords = ~amd64  
    - Flatpak for some desktop apps  
    - Bloat  
    - MTP and iOS support
